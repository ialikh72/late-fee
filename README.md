# Late Fee Calculator (C# Console App)

A simple C# console program that calculates a late fee (tax) based on how many days a payment is late, then shows the total amount to pay.

## 📌 Late Fee Rules

- 0 days late → $0 tax  
- 1–3 days late → $1 tax  
- 4–6 days late → $2 tax  
- More than 6 days late → $3 tax  

## ▶️ How It Works

1. The user enters their monthly fee.
2. The user enters how many days late the payment is.
3. The program calculates the tax based on the days late.
4. The program prints the tax and the total fee.
